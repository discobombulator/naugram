## Запуск проекта

### Требования
- JDK 17+
- PostgreSQL
- Gmail аккаунт с паролем приложения

### Настройка

Укажите следующие параметры в `application.properties`:

```properties
# База данных
spring.datasource.url=jdbc:postgresql://localhost:5432/ваша_бд
spring.datasource.username=ваш_пользователь
spring.datasource.password=ваш_пароль

# Почта для отправки кодов
mail.user=ваша_почта@gmail.com
mail.password=пароль_приложения_gmail
```

> **Важно:** Для `mail.password` используйте пароль приложения Gmail,
> а не основной пароль аккаунта. Получить его можно в настройках
> Google аккаунта → Безопасность → Пароли приложений.

## Функциональность

- Регистрация и авторизация пользователей
- Двухфакторная аутентификация через email
- Сессии с автопродлением при активности
- Выход из аккаунта
