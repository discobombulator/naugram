## Технологии

- Java 21
- Spring Boot
- Spring Security
- Spring Data JPA
- Thymeleaf
- PostgreSQL
- JavaMail (Gmail SMTP)

---

## Запуск проекта

### Требования
- JDK 17+
- PostgreSQL
- Gmail аккаунт с паролем приложения

### Настройка

Укажите следующие параметры в `application.properties`:

```properties
# База данных
spring.datasource.url=jdbc:postgresql://localhost:5432/ваша_бд
spring.datasource.username=ваш_пользователь
spring.datasource.password=ваш_пароль

# Почта для отправки кодов
mail.user=ваша_почта@gmail.com
mail.password=пароль_приложения_gmail
```

> **Важно:** Для `mail.password` используйте пароль приложения Gmail,
> а не основной пароль аккаунта. Получить его можно в настройках
> Google аккаунта → Безопасность → Пароли приложений.

## Функциональность

- Регистрация и авторизация пользователей
- Подтверждение аккаунта через email
- Двухфакторная аутентификация через одноразовый код на email
- Сессии с автопродлением при активности
- Выход из аккаунта

### Настройки пользователя

- Смена пароля
- Проверка текущего пароля перед сменой
- Включение и выключение двухфакторной аутентификации
- Выбор языка интерфейса
- Сохранение выбранного языка пользователя в базе данных

### Локализация

- Поддержка русского и английского языков
- Переключение языка через настройки пользователя
- Локализация главной страницы
- Локализация страницы чата
- Использование отдельных `messages_ru.properties` и `messages_en.properties`

### Профиль пользователя

- Редактирование имени и фамилии
- Изменение тега пользователя
- Проверка уникальности тега перед сохранением
- Добавление даты рождения
- Добавление описания профиля
- Загрузка пользовательской аватарки
- Отображение аватарки пользователя в профиле
- Отображение аватарки собеседника в чате

### Главная страница

- Отображение списка доступных пользователю чатов
- Переход в выбранный чат
- Панель настроек
- Панель редактирования профиля
- Приветственный экран при отсутствии выбранного чата

### Чаты

- Отдельная страница конкретного чата
- Возврат из чата на главную страницу
- Отправка текстовых сообщений через WebSocket/STOMP
- Отображение истории сообщений при открытии чата
- Разделение сообщений текущего пользователя и собеседника по разным сторонам
- Отображение аватарки пользователя рядом с сообщением
- Открытие мини-профиля пользователя по клику на аватарку сообщения
- Отображение в мини-профиле:
  - аватарки
  - имени пользователя
  - тега пользователя
  - описания профиля
  - даты рождения

### Медиафайлы

- Локальное хранение медиафайлов на диске
- Хранение публичных путей к медиафайлам в базе данных
- Загрузка пользовательских аватарок
- Отправка изображений в чат
- Отправка видео в чат
- Отправка нескольких медиафайлов одним сообщением
- Добавление подписи к медиа-сообщению
- Предпросмотр медиа перед отправкой
- Добавление дополнительных файлов в редакторе медиа
- Удаление выбранного файла из набора перед отправкой
- Отображение медиа-сообщений в виде сетки
- Просмотр изображения или видео на отдельной странице
- Возврат из просмотра медиа обратно в чат

---
### Физическая структура файлов

Медиафайлы разделяются по назначению:

| Директория | Назначение |
|---|---|
| `E:/NauGramMedia/avatars/users/{userId}/` | Аватарки пользователей |
| `E:/NauGramMedia/chats/{chatId}/` | Медиафайлы конкретного чата |
