## Технологии

- Java 21
- Spring Boot
- Spring Security
- Spring Data JPA
- Thymeleaf
- PostgreSQL
- JavaMail (Gmail SMTP)

---

## Запуск проекта

### Требования
- JDK 17+
- PostgreSQL
- Gmail аккаунт с паролем приложения

### Настройка

Укажите следующие параметры в `application.properties`:

```properties
# База данных
spring.datasource.url=jdbc:postgresql://localhost:5432/ваша_бд
spring.datasource.username=ваш_пользователь
spring.datasource.password=ваш_пароль

# Почта для отправки кодов
mail.user=ваша_почта@gmail.com
mail.password=пароль_приложения_gmail
```

> **Важно:** Для `mail.password` используйте пароль приложения Gmail,
> а не основной пароль аккаунта. Получить его можно в настройках
> Google аккаунта → Безопасность → Пароли приложений.

## Функциональность

- Регистрация и авторизация пользователей
- Подтверждение аккаунта через email
- Двухфакторная аутентификация через email
- Возможность включить или выключить двухфакторную аутентификацию в настройках
- Смена пароля пользователя через настройки аккаунта
- Сессии с автопродлением при активности
- Выход из аккаунта
- Главная страница приложения с боковой панелью чатов
- Редактирование профиля пользователя:
  - имя
  - фамилия
  - тег пользователя
  - дата рождения
  - описание / биография
- Проверка уникальности тега пользователя перед сохранением
- Локализация интерфейса на русский и английский языки
- Сохранение выбранного языка пользователя в базе данных
- Автоматическое восстановление выбранного языка после повторного входа или перезапуска приложения
- Уведомления об успешных и неуспешных операциях
